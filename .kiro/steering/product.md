# KaraokeVerse

VR karaoke application built with IWSDK for Meta Quest devices. Users create a profile, select themed karaoke rooms, and sing along to YouTube videos with a grabbable microphone.

## Core Features

- Profile creation and persistence
- 5 themed karaoke rooms (Anime Tokyo, K-pop Seoul, Bollywood Mumbai, Hollywood LA, Taylor Swift Broadway)
- Grabbable 3D microphone with visual feedback
- YouTube video playback on virtual screens
- Song search and selection by room theme
- Full VR controller interaction with ray-based UI

## User Flow

1. Profile check/creation → 2. Room selection → 3. Enter themed room → 4. Song search → 5. Sing with video playback
